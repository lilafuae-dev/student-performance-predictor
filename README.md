# Student Performance Predictor Using Machine Learning

## Overview
This project uses Python and Machine Learning to predict student academic grades based on key factors such as:

- Hours studied
- Sleep duration
- Phone usage

The goal of the project is to explore how daily habits may influence academic performance using predictive modeling.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Model

A Linear Regression model was trained using sample student data to identify patterns between lifestyle factors and grades.

---

## Example Prediction

Input:

- Study Hours: 6
- Sleep Hours: 7
- Phone Usage: 2

Predicted Grade:

87%

---

## Key Findings

- Increased study time generally improved predictions
- High phone usage lowered predicted grades
- Balanced sleep had a positive effect

---

## Purpose

This project was created as an introductory exploration into Artificial Intelligence and Machine Learning, demonstrating how data can be used to make predictions and support decision-making.
